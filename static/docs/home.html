pure javascript json and files storage engine

pouchlite draws inspiration from tools like

    mongoose mongodb sqlite db
data in pouchlite is stored in the file system,we try and make it user first storage engine

FEATURES

embedable

think of sqlite but pouchlite,easily integrates with any backend framework or fullstack

ultra fast

performance is key,reads and writes are super fast

orm

if you know mongoose ,pouchlite impliments mongoose like querries and api

Dx

simple Dx ,you will love it

sync

sync local with remote

schema

inbuilt schema validator

collections

documents are stored in collections
