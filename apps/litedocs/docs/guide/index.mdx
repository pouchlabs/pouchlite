# what is pouchlite

is a document and files storage engine,thats is small for building offline-first applications,caching document,under the hood its secure storage solution for javascript devs,its opensource, it serves as an alternative to other embedable storage solutions i.e sqlite and the rest.

## inspiration

pouchlite is inspired by tools like sqlite,pouchdb,redis,mongodb,drizzle,mongoose.
these tools inspired me into creating pouchlite .
